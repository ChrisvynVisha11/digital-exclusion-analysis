# Digital Exclusion & Qualification Attainment in England

Analysis of the Next Steps (LSYPE) longitudinal dataset 
investigating structural predictors of digital exclusion 
among young people in England.

## Methods
- Ordinal logistic regression with odds ratios
- Elastic net and random forest classification
- Mediation analysis (parental education as mediator)
- Class imbalance handled using recall as primary metric
- Feature standardisation and complete-case analysis

## Key Findings
- Parental education and neighbourhood deprivation 
  strongest predictors across all models
- Adolescents without home internet at 13 had 28% lower 
  odds of higher qualification attainment
- Parental education partially mediates digital exclusion's 
  effect on qualification attainment

## Tools
Python, pandas, scikit-learn, statsmodels, 
matplotlib, seaborn
